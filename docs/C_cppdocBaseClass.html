Class cppdocBaseClass

This is the base class from which the two primary cppdoc classes are derived. This class declares the member data that is common to both classes.

Created by Digital Reality (11/10/2002)

Member Data
public string name - Name of the object
public string parent - Name of the object's parent
public bool hasParent - True if the object has a parent
public string creator - Creator's name
public string created - Date of creation
public vector<string> modifiers - List of modifiers' names
public vector<string> modified - List of modification dates
public string desc - Description of the object
public string docObjectType - What kind of object this is

Children classes: cdGlobal, cdClass, cdFunction
